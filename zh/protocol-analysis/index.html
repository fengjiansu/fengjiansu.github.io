  1. Protocol Analysis ( also referred to as network analysis ) is the process of tapping into the network communications system, capturing packets that cross the network, gathering network
    statistics, and decoding the packets into readable form.
    协议分析(也称为网络分析)是进入网络通信系统捕获穿行在网络中的数据(报文),收集网络统计信息,将数据包解码为可阅读形式的过程
  2. broadcast address 广播地址: the all-ones address for a network or subset,which provides a way to send the same information to all interfaces
    on a network
    网络或子网的全1地址,这种地址提供一种把相同信息发送给网络上所有网卡的方法
  3. address masquerading 地址伪装: a method of mapping many internal(i.e private)nonroutable address to a single external(i.e,public)ip address for the pirpose of sharing a single internet connection-also referred to as address hiding
    一种把多个内部的(即私有的)非路由地址映射到单个外部(即公用的)IP地址的方法.其目的是为了共享单个Internet连接,又称位地址隐藏
  4. anycast address a type of ordinary address in IPv6 that can be assigned to more than one host or interface.Packets pointed to an anycast address are delivered to the holder of that address nearest to the sender in terms of routing distance.An anycast address does not apply to IPv4
    是IPv6中的一种地址类型,可以分配给多个主机或接口.指向任播地址的数据包,可以根据路由距离传递给离发送方最近的那个地址.任播地址不能用于ipv4
  5. connectionless protocol:a protocol that simply sends datagrams without establishing ,managing,or otherwise handling a connection between sender and receiver;UDP is a connectionless Transport layer protocol
    一种网络协议,只负责发送数据包,创建 管理或处理发送方和接收方之间的连接.udp就是一种无连接协议
  6. connection-oriented transport mechanisms : A transport layer protocol that establishs a session between two computers,provides error checking and guarantees delivery
    一种传输层协议,它在两个计算机之间创建一个会话,提供错误检测,确保成功传输.
  7. jumbogram: a specification for allowing very large(beyond 4 gigabyte)packets to be transported using ipv6 used only in special circumstance such as on large backbone
    对允许ipv6传送超大(大于4g字节)数据包的规范说明 仅仅用于特殊环境下比如大型主干路由
  8. link MTU 链路最大传输单元:this is the MTU capacity of a specific link within a path MTU. the smallest link mtu determine the MTU size of ipv6 packets for the path
    某个路径MTU中特定链路的MTU容量.最小的链路MTU决定了路径中ipv数据包的MTU
  9. precedence :a definition of priority for an ip packet.routers may process higher-priority packets before lower-priority packets when a router queue is congested
    ip数据包优先权的一个定义.当路由器队列被阻塞时路由器可以在优先级低的数据包之前优先处理较高优先级的数据包

  3. 为什么错误太多不好
    网络上的过度错误使通常可用的带宽完全无法使用。因此，网络的整体有效利用率通常会下降，即使网络的测量利用率实际上可能会增加。因为错误仍然消耗带宽，即使它们没有传达任何有用的内容，它们也被认为是"带宽窃取者"，应该尽快定位和纠正。因此，网络监视器和协议分析器通常设置报警阈值，此时错误会消耗 2% 的网络带宽。虽然在没有基础病理的情况下偶尔会达到这个水平，但如果达到这个水平有任何规律性，它通常表明网络上出了问题 同样，由于有线电视段上的所有电台都必须阅读广播流量，在必须回复或回复其内容的偶然机会上，过度广播对网络利用也有同样的影响——它们窃取了本来可以使用的带宽，而没有实际的好处。与错误一样，广播流量相同的 2% 阈值是设置闹钟的好地方。事实上，有一个严重的病理学，称为"广播风暴"（通常是由某种网络接口故障引起的），其中所有可用的网络带宽都被虚假广播所消耗。单单这基本上使网络无法使用，定位和修复这样的问题是一个高度优先事项 ！
